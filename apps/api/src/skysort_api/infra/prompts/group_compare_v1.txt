評価観点:
- 機首、主翼、尾翼、車輪、エンジン、翼端が不自然に切れていないか
- ピント、ブレ、流し撮りの芯、機体のディテールが残っているか
- 機体姿勢、角度、背景、障害物、陽炎、構図の余白が良いか
- 似た構図の中で残す価値がある差分があるか
- best は必ず1枚だけ、keep は残す価値がある複数枚、reject は候補から外すべき失敗カットとして区別する

problem_tags は必要に応じて以下から選び、該当がなければ空配列にしてください:
